产品介绍 / Introduction

快邮宝是快邮速递快邮速递整合各方资源，与美国邮政USPS合作而开发的针对于跨境电商量身打造价格优、时效好的优质渠道。

递送范围 / Delivery Range

美国全境

  • 重量限制 / Weight Restriction

    0-30KG

  • 尺寸限制 / Size Restriction

    长+2*（宽+高）≤270cm

产品优势 / Product Advantage

  • 可走带电

  • 可退税

  • 支持USPS物流查询

  • 清关能力强

  • 时效好

  • 多元化定制服务

资费价格 / Tariff Price

1、0 ≤ 重量 ≤ 250g：0.08元/克+12元/件，不足50g按50g算

2、250g＜重量 ≤ 2kg：0.08元/克+10元/件

3、2kg＜重量 ≤ 3kg：起重60元/500g，续重30元/500g

4、3kg＜重量 ≤ 30kg：起重40元/500g，续重30元/500g

注：
 包裹长超过68.58cm，或者宽超过43.18cm，或者高超过43.18cm，需要加收30元的特殊处理费；
包裹长小于15.24cm，且宽小于0.635cm，且高小于7.62cm，也需要加收30元的特殊处理费；

退税流程 / Tax Refund Process

如需办理退税，用户需先完成企业和商品等相关信息的备案，详情请联系客服

其他说明 / Other Instructions

本渠道只提供深圳仓库出仓前拦截服务，客户必需在我司系统内做拦截动作，否则我司将不予以受理。

禁运物品 / Prohibited Articles

任何形式的仿牌、纯电、移动电源、刀具、液体等任何航空违禁品
